# CURRICULUM AND MAP — Whips: 55-Level Jungle Math Game

## 1. Mastery Rules

### 1.1 Skill Tag System
Every task is tagged with 1-3 skill tags from a controlled vocabulary. Mastery is tracked per skill tag independently.

### 1.2 Mastery Levels
| Level | Name | Criteria | Visual |
|-------|------|----------|--------|
| 0 | New | No attempts | Gray star |
| 1 | Introduced | Completed warmup diagnostic | Bronze star |
| 2 | Practiced | 3+ correct answers in practice | Silver star |
| 3 | Proficient | Streak of 3 at difficulty >= 2, total 5+ correct | Gold star |
| 4 | Mastered | Boss challenge passed with streak of 5 | Platinum star |

### 1.3 Spaced Review
- Skills at level 2+ enter the review queue
- Review intervals: 1 day → 3 days → 7 days → 14 days → 30 days
- Failed review resets interval to 1 day (does not reduce mastery level)
- Practice arena shows "due for review" topics with a leaf icon

### 1.4 Difficulty Scaling
- Tasks have difficulty 1-5
- Adaptive mode: starts at difficulty matching mastery level, adjusts ±1 based on streak
- Manual mode: player chooses difficulty tier (Sprout / Sapling / Tree)
